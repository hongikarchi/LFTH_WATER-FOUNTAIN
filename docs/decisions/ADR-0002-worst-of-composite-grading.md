# ADR-0002: Worst-of composite water-quality grading

- **Status**: Accepted
- **Date**: 2026-05-19

## Context

환경정책기본법 시행규칙 별표 1 (하천 생활환경기준)은 측정 항목별로 등급(Ia~VI)을 규정한다. 측정소 하나의 "종합 수질 등급"을 어떻게 산출할 것인가에 대한 표준 합의된 단일 공식은 없다(보통 항목별 등급을 모두 함께 표기). 우리 UI는 측정소별로 "하나의 등급"을 보여줘야 사용자가 빠르게 좋다/나쁘다를 판단할 수 있다.

또한 다음 항목은 별도 처리가 필요:
- **TOT_N (총질소)**: 별표 1 하천 기준에 포함되지 않음 (호소 기준에만 존재). 우리 측정소(탄천/중랑천/안양천/선유)는 하천이므로 호소 기준을 적용하면 잘못된 인상 전달.
- **PHNL (페놀), CN (시안)**: 사람의 건강보호기준에 속하는 독성 물질. 검출/불검출 자체가 중요하지 등급으로 "약간 나쁨" 같은 표현은 부적절.

## Decision

1. **종합 등급 = pH / DO / TOC / T-P 4 항목 중 worst (가장 나쁜 등급)**.
   - 가장 보수적 해석 → 한 항목이라도 나쁘면 전체가 나쁘다고 표시
   - `null` 항목은 skip (점검중인 센서 때문에 항상 worst가 되는 것 방지)
   - 4 항목이 모두 `null`이면 종합 등급 = `'unmeasurable'` 센티넬 → UI는 회색 카드 + "측정불가" 텍스트
2. **T-N은 등급 산정에서 제외**. UI에서는 측정값을 계속 표시하되 등급 색/뱃지는 부여하지 않음.
3. **페놀, 시안은 별도 표기**. 등급에 포함하지 않고 카드에 별도 pill로 `검출` / `불검출` / `—` 텍스트 우선 표시(색은 보조). Phase 1에서는 `value > 0 → 검출` 단순 규칙 사용. 별표 1의 정확한 검출한계(페놀 ≈ 0.005 mg/L, 시안 ≈ 0.01 mg/L) 적용은 Phase 2에서 보정.

## Consequences

**긍정**:
- 사용자가 한 측정소를 보면 "지금 좋다/나쁘다" 즉시 판단 가능
- 점검중 센서 때문에 등급이 부당하게 떨어지는 일 없음
- 독성과 일반 수질이 분리되어 메시지가 흐려지지 않음

**부정**:
- "왜 종합 등급이 IV인가?"를 사용자가 알려면 LatestTable의 항목별 색 dot을 봐야 함 (큰 카드에 항목별 등급 모두 표시하지는 않음 → 정보 밀도 trade-off)
- 페놀/시안 검출 임계값이 Phase 1에서 거칠다 — 실제 측정 노이즈 floor가 0보다 약간 높을 수 있어 false positive 가능. Phase 2에서 별표 1 검출한계로 보정 예정
- T-N을 평가에서 빼면 부영양화 정보가 누락된 것 아닌가 우려 — 측정값 표시는 유지하므로 정보 자체는 잃지 않음
