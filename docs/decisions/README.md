# Architecture Decision Records (ADR)

이 폴더는 프로젝트의 의도가 코드에 드러나지 않는 "왜 이렇게 했는가"를 기록한다.
새 ADR을 추가할 때는 번호를 증가시키고 표에 한 줄 추가한다.

## 포맷

[Nygard 형식](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions): Title / Status / Context / Decision / Consequences.
